## About this website

This website is the online home of the US LHC Users Association (USLUA). It serves as a central hub for information about our organization, upcoming events, and resources for the US community working on LHC experiments.

### Technology

This website is built using [Hugo](https://gohugo.io/), a fast and modern static site generator, with the [Blowfish](https://blowfish.page/) theme.

### Contributing

We welcome contributions from the community! If you'd like to suggest improvements, report issues, or contribute content, please see our [Contributing Guidelines]({{< param "githubRepo" >}}/blob/main/CONTRIBUTING.md), part of the [GitHub repo]({{< param "githubRepo" >}}) that builds this website.

### Maintenance

This website is maintained by the US LUA Executive Committee and volunteers from the community.

### Contact

For questions about this website or to report technical issues, please create an issue on the [GitHub repo]({{< param "githubRepo" >}}).
